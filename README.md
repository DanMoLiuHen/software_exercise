## 运行
1. 安装包`npm install`
2. 运行`npm run serve`
3. build `npm run build`

## 说明
views中是所有的视图
```
├─calendar
├─cash
├─eleven
├─four
├─home
├─layout
│  └─components
│      ├─asidebar
│      └─headbar
├─login
├─nine
├─sales
├─seven
├─ten
├─thirteen
├─triangle
└─twelve
```
在views中home与layout控制整体布局（包括侧边栏等），`route`中控制路由
